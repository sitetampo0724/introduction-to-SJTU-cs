# [计算机科学与技术](https://i.sjtu.edu.cn/jxzxjhgl/pyjhxxcx_cxPyjhxxIndex.html#)
计算机科学与技术专业是首批国家级一流本科专业建设点，所属学科为国家一级重点学科、双一流建设学科。师资力量雄厚，国家级人才比例近半。学科核心专业方向涵盖新一代人工智能、云原生计算系统、领域操作系统与系统软件、网络空间安全与密码支撑体系、物联网与端侧计算、理论计算机科学创新方法等。在课程体系建设中，大幅采用国产软硬件最新成果和企业案例，率先在课程中引入国产昇腾AI芯片、鸿蒙操作系统等自主核心技术相关的教学内容，与华为、腾讯、阿里、商汤等龙头企业合作，共建学生实践基地，践行产教融合的教学方法与实践培养。学生竞赛成绩斐然，三次获得ACM国际大学生程序设计竞赛全球总冠军，数模国赛32年历史上唯一实现三大奖项大满贯高校，美国数模竞赛特等奖数目累计世界第一。
