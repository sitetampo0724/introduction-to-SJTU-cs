# 学院总体概况
> Pioneering Computing Excellence, Shaping Future Innovators

上海交通大学计算机学院于2025年5月11日正式成立，标志着该学院进入全新发展阶段。学院设有计算机科学与技术、信息安全、软件工程三个国家级一流本科专业，并设有永强班、ACM班、John班等特色班级，旨在因材施教，为不同层次和特长的学生提供优质教育资源。学院与华为、阿里、中国移动、中国电信等知名企业紧密合作，建立联合实验室和人才培养基地，推动科研成果转化为实际生产力。

# 电院分流机制
- **机制**: 80%综合评价成绩+20%分流考
- **内容**: 分流考涉及数学和语义逻辑，各占一半
- **排名**: 大概入学三个月后结束分流工作，从高到低选专业
